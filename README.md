
# RMDutil

## Installation

you can install the development version from
[GitHub](https://github.com/) with:

``` r
# install.packages("devtools")
devtools::install_github("Larsdegroot/RMDutil")
```

## Example

``` r
library(RMDutil)
```

``` r
data("data_covid")
```

dim(data_covid)
